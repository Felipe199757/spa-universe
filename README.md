
# Spa Universe

this is a project about a Single Page Aplicattion(SPA)
I learned how to create a page that is not updated when changing the address within the site
## Functionalities
- destructuring
- preventDefault()
- DOM (window, location)
- pushState()
- fetch()
- classes
- onclick()
